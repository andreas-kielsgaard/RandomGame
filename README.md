# Cosmic Fetch Quest: The Game

A browser-playable Phaser prototype about fetching impossible ingredients across a glowing psychedelic cosmos.

## Install

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

Vite will print a local URL, usually `http://localhost:5173`.

## Prototype Status

Iteration 1 sets up the game shell with Vite and Phaser 3. Level 1 is playable with generated placeholder textures, a player, gravity, platforms, camera follow, one collectible ingredient, and one NPC interaction. Levels 2 through 5 are stubbed in `src/data/levels.js` for future expansion.

## Controls

- Move: `A` / `D` or Left / Right Arrow
- Jump: `W`, Up Arrow, or Space
